Om
inject
Operation

Inserts a Program before each Operand of another. More...

Input
{injector}{program}
Output
Dequotes {program}, injects the dequoted {injector} between the Program and each of its Operands, then quotes the result.
Examples
  • inject{quote}{copy}{A}

    {{{A}}{{A}}}
